# How to use

1 - put all songs inside a folder

2 - use the "Files" menu and "select folder" option 

3 - choose your music folder

4 - use the play button to play

5 - use the pause button to pause.

6 - If you need to unpause, use the play button or click the pause button again.

7 - Use the forward button to go to the previous song

8 - And the next button to go to the next song
